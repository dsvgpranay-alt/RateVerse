# 💱 RateVerse

### Real-Time Rates, Instant Conversions

RateVerse is a modern currency converter web application that allows users to convert currencies using live exchange rates. The application provides a clean and responsive interface, dynamic currency selection, country flag integration, and instant conversion results.

---

## 🚀 Features

- 🌍 Convert between multiple international currencies
- ⚡ Real-time exchange rate conversion
- 🚩 Dynamic country flag updates
- 🔄 Easy currency selection
- 📱 Responsive design for all devices
- 🎯 Fast and accurate conversion results

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API

## 🎮 Usage

1. Enter the amount to convert.
2. Select the source currency.
3. Select the target currency.
4. Click the **Convert** button.
5. View the converted amount instantly.
